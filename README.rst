prize
=========

A tool to generate user cards and pickup a winner for prize randomly.

一个抽奖工具，支持输入内容，然后随机抽取一个幸运个体。

Package hosted on [`pypi`_] and powered by [`pypi-seed`_]

Installation / 安装
--------------------------

::

    pip install prize
